## Installing sudoku-hero

To install sudoku-hero, follow these steps:

```bash
git clone https://github.com/2kabhishek/sudoku-hero
cd sudoku-hero/src
make
```

This will generate a `sudoku-hero` binary in the bin folder/

## Using sudoku-hero

```bash
# After make
cd bin/
./sudoku-hero
```
